## About

자기 소개 페이지입니다. 이력서를 겸합니다.

[Coalery/about](https://github.com/Coalery/about.git) 레포를 기반으로 제작하였습니다.

### Prerequisites

- Node.js v20.11.1

### Install

```shell
git clone https://github.com/cpprhtn/about.git
cd about
npm install
```

### Run in dev

```shell
npm run dev
```

### Build

```shell
npm run build # output: /out
```
